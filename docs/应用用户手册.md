# 应用用户手册

## 概述

iBreeze 桌面应用是 AI 公司运营的核心工具，提供与 AI 职员协作、任务管理、知识库等功能。

## 安装

### Windows
1. 下载安装包
2. 双击运行安装程序
3. 按照向导完成安装

### macOS
1. 下载 DMG 文件
2. 双击打开并将应用拖入 Applications
3. 首次运行需要在系统偏好设置中允许

### Linux
```bash
# Debian/Ubuntu
sudo dpkg -i ibreeze_*.deb

# Fedora/RHEL
sudo rpm -i ibreeze_*.rpm
```

## 首次启动

1. 启动应用
2. 选择「新建公司」或「加入已有公司」
3. 填写公司信息
4. 完成初始化设置

## 主界面

### 侧边栏导航

- **仪表板**: 系统概览和快捷操作
- **会话**: 与 AI 职员的对话
- **任务**: 任务管理和跟踪
- **工作区**: 文档协作空间
- **设置**: 个人和系统设置

## 会话功能

### 创建新会话

1. 点击「新建会话」按钮
2. 选择参与的 AI 职员
3. 输入会话标题（可选）
4. 点击「创建」

### 发送消息

1. 在消息输入框输入内容
2. 按 Enter 或点击「发送」
3. AI 职员会自动回复

### 消息功能

- **复制**: 点击消息旁的复制按钮
- **收藏**: 点击星标收藏重要消息
- **回复**: 引用特定消息回复

### 会话管理

- **搜索**: 使用顶部搜索栏查找历史会话
- **归档**: 右键会话选择「归档」
- **删除**: 右键会话选择「删除」

## 任务管理

### 创建任务

1. 进入「任务」页面
2. 点击「新建任务」
3. 填写任务信息：
   - 标题（必填）
   - 描述（可选）
   - 优先级（低/中/高）
   - 指派给（可选）
4. 点击「保存」

### 任务状态

- **待处理**: 尚未开始
- **进行中**: 正在执行
- **已完成**: 任务完成
- **已取消**: 任务取消

### 任务视图

- **列表视图**: 以列表形式查看
- **看板视图**: 按状态分栏显示
- **时间线**: 查看任务时间安排

## 工作区

### 创建工作区

1. 进入「工作区」页面
2. 点击「创建工作区」
3. 输入工作区名称
4. 点击「创建」

### 文档协作

1. 进入工作区
2. 点击「新建文档」
3. 编辑文档内容
4. 自动保存

### 版本控制

- 查看文档历史版本
- 恢复到历史版本
- 比较版本差异

### 审批流程

1. 提交文档审批
2. 审批人收到通知
3. 审批人审核并批准/拒绝
4. 文档状态更新

## AI 职员

### 查看 AI 职员列表

1. 点击侧边栏「AI 职员」
2. 查看所有可用的 AI 职员
3. 查看每个职员的技能和状态

### 与 AI 职员协作

1. 选择一个 AI 职员
2. 发起会话
3. 描述你的需求
4. AI 职员会协助完成任务

### 管理 AI 职员技能

1. 进入 AI 职员详情
2. 查看已安装技能
3. 安装新技能
4. 卸载不需要的技能

## 知识库

### 添加知识文档

1. 进入「知识库」页面
2. 点击「添加文档」
3. 输入标题和内容
4. 添加标签（可选）
5. 点击「保存」

### 搜索知识

1. 使用搜索栏输入关键词
2. 按标签筛选
3. 按类型筛选
4. 查看搜索结果

### 知识共享

- 将知识文档共享给特定部门
- 设置访问权限
- 导出知识文档

## 设置

### 个人设置

- 修改头像
- 更新个人信息
- 修改密码

### 通知设置

- 邮件通知
- 桌面通知
- 通知频率

### 外观设置

- 深色模式/浅色模式
- 字体大小
- 语言设置

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + N` | 新建会话 |
| `Ctrl + K` | 快速搜索 |
| `Ctrl + S` | 保存 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + /` | 显示快捷键 |

## 数据同步

- 本地数据自动同步到云端
- 离线状态下可继续工作
- 联网后自动同步

## 备份与恢复

### 自动备份

- 系统自动备份重要数据
- 备份频率：每日

### 手动备份

1. 进入设置
2. 点击「备份」
3. 选择备份位置
4. 等待完成

### 恢复数据

1. 进入设置
2. 点击「恢复」
3. 选择备份文件
4. 确认恢复

## 常见问题

### Q: 应用无法启动？
A: 检查系统要求，尝试重新安装。

### Q: 数据丢失了怎么办？
A: 检查回收站，或从备份恢复。

### Q: 如何联系支持？
A: 通过应用内「帮助」菜单联系客服。

### Q: 如何更新应用？
A: 系统会自动提示更新，或手动检查更新。
