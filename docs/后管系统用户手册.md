# 管理后台用户手册

## 概述

iBreeze 管理后台是用于管理公司、用户、技能和目录的 Web 界面。

## 登录

1. 打开浏览器访问管理后台地址
2. 输入用户名和密码
3. 点击「登录」

## 首页仪表板

登录后显示以下信息：
- 系统概览（用户数、技能数、活跃会话数）
- 最近操作日志
- 系统状态

## 用户管理

### 查看用户列表

1. 点击左侧菜单「用户」
2. 查看所有用户信息
3. 支持按用户名、邮箱搜索

### 创建用户

1. 点击「添加用户」按钮
2. 填写用户信息：
   - 用户名（必填）
   - 邮箱（必填）
   - 密码（必填）
   - 角色（admin/editor/viewer）
3. 点击「保存」

### 编辑用户

1. 在用户列表中点击「编辑」
2. 修改用户信息
3. 点击「保存」

### 禁用/启用用户

1. 在用户列表中切换「状态」开关
2. 确认操作

## 技能管理

### 查看技能列表

1. 点击左侧菜单「技能」
2. 查看所有技能信息
3. 支持按分类筛选

### 上传技能

1. 点击「上传技能」按钮
2. 选择技能 ZIP 文件
3. 填写技能信息：
   - 名称（必填）
   - 版本号（必填）
   - 分类（必填）
   - 描述（可选）
   - 兼容性要求（可选）
4. 点击「上传」

### 启用/禁用技能

1. 在技能列表中切换「状态」开关
2. 禁用的技能将对所有用户不可用

### 紧急禁用

在安全事件发生时：
1. 进入技能详情
2. 点击「紧急禁用」按钮
3. 确认操作
4. 技能立即对所有用户不可用

## 目录发布

### 创建发布版本

1. 点击左侧菜单「目录」
2. 点击「创建发布」
3. 系统自动生成 Manifest
4. 填写发布说明（可选）
5. 点击「保存为草稿」

### 发布目录

1. 在发布列表中选择草稿版本
2. 点击「发布」
3. 确认操作
4. 目录版本生效

### 查看发布历史

1. 在目录页面查看所有发布版本
2. 查看每个版本包含的技能列表
3. 查看发布时间和状态

## 审计日志

### 查看审计日志

1. 点击左侧菜单「审计日志」
2. 查看所有操作记录
3. 支持按时间、用户、操作类型筛选

### 日志详情

点击日志条目查看：
- 操作时间
- 操作用户
- 操作类型
- 资源信息
- IP 地址
- 详细信息

## 系统设置

### 基本设置

- 系统名称
- 系统描述
- 时区设置

### 安全设置

- 会话超时时间
- 密码策略
- API 访问控制

### 通知设置

- 邮件通知配置
- 告警阈值设置

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + K` | 快速搜索 |
| `Ctrl + N` | 创建新项目 |
| `Esc` | 关闭弹窗 |

## 常见问题

### Q: 忘记密码怎么办？
A: 联系系统管理员重置密码。

### Q: 如何批量导入用户？
A: 使用「导入用户」功能，上传 CSV 文件。

### Q: 技能上传失败？
A: 检查 ZIP 文件格式和大小限制（最大 50MB）。

### Q: 如何查看系统日志？
A: 通过「审计日志」页面查看操作记录。
