# iBreeze 打包方式文档

## 技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| 桌面框架 | Tauri 2.x | Rust + WebView |
| 前端 | React 19 + TypeScript + Vite | |
| 后端 | Python 3.12+ (uv) | Sidecar 进程，UDS 通信 |
| 数据库 | SQLite | `/tmp/acos.db` |
| 通信协议 | JSON-RPC 2.0 | Unix Domain Socket |

## 打包方式

### 开发模式

```bash
cd apps/desktop
npm run tauri dev
```

- 前端热更新，后端独立运行
- Sidecar 由 Rust 自动启动（`src-tauri/src/lib.rs`）

### 生产构建

```bash
cd apps/desktop
npm run tauri build
```

构建产物：

```
apps/desktop/src-tauri/target/release/bundle/
├── macos/
│   └── iBreeze.app          # macOS 应用包
└── dmg/
    └── iBreeze_0.1.0_aarch64.dmg  # 安装镜像
```

### 安装

**方式一：拖拽安装**

打开 `.dmg` 文件，将 `iBreeze.app` 拖入 `Applications` 文件夹。

**方式二：命令行安装**

```bash
cp -R src-tauri/target/release/bundle/macos/iBreeze.app /Applications/
```

## Sidecar 进程管理

### 启动流程

1. Rust 端通过 `Command::new("uv").arg("run").arg("python").arg("-m").arg("acos.app")` 启动 sidecar
2. Sidecar 在 `/tmp/acos.sock` 创建 Unix Domain Socket
3. Rust 端等待 socket 就绪（最多 10 秒）
4. 前端通过 `invoke('sys_rpc_call', ...)` 与 sidecar 通信

### 进程清理

**正常退出**：点击窗口关闭按钮 → Tauri `RunEvent::Exit` → Rust 端 `kill_sidecar()` 杀掉整个进程树

**强制退出**：`kill -9` 或 Activity Monitor 强制退出 → Sidecar 通过 `IBREEZE_APP_PID` 环境变量监控主进程，检测到主进程死亡后自动退出（2 秒检测间隔）

## 构建配置

### tauri.conf.json

```json
{
  "productName": "iBreeze",
  "version": "0.1.0",
  "identifier": "com.ibreeze.desktop",
  "build": {
    "beforeBuildCommand": "npm run build",
    "frontendDist": "../dist"
  },
  "bundle": {
    "targets": "all",
    "icon": ["icons/icon.icns"]
  }
}
```

### Rust 依赖 (Cargo.toml)

```toml
[dependencies]
tauri = { version = "2", features = [] }
serde = { version = "1", features = ["derive"] }
serde_json = "1"
tokio = { version = "1", features = ["rt-multi-thread", "net", "io-util", "macros"] }
```

## 开发环境依赖

| 组件 | 最低版本 | 说明 |
|------|----------|------|
| Node.js | 18+ | 前端构建 |
| Python | 3.12+ | Sidecar 后端 |
| uv | latest | Python 依赖管理 |
| Rust | stable | Tauri 编译 |
| Xcode CLI Tools | - | macOS 必需 |

### 安装步骤

```bash
# 1. 克隆项目
git clone <repo-url>
cd ibreeze

# 2. 安装前端依赖
cd apps/desktop && npm install

# 3. 安装 Python Sidecar 依赖
cd ../../sidecar && uv sync

# 4. 启动开发模式
cd ../apps/desktop && npm run tauri dev
```

## 常见问题

### Q: Tauri 构建失败

确认已安装 Rust 工具链和系统依赖：

```bash
xcode-select --install
rustup update
```

### Q: Sidecar 连接失败

确保 sidecar 已启动且 socket 文件存在：

```bash
ls -la /tmp/acos.sock
```

### Q: 应用关闭后 sidecar 残留

```bash
# 手动清理
pkill -9 -f "acos.app"
rm -f /tmp/acos.sock
```

### Q: 重新构建前清理

```bash
pkill -9 -f "ibreeze-desktop"
pkill -9 -f "acos.app"
rm -f /tmp/acos.sock
rm -rf "/Applications/iBreeze.app"
```
