# 本地部署文档

## 环境要求

### 必需软件
- **Node.js** >= 20.x
- **Python** >= 3.12
- **Rust** >= 1.75 (用于桌面核心)
- **PostgreSQL** >= 16

### 可选软件
- **Docker** & **Docker Compose** (推荐)
- **pnpm** (包管理器)

## 快速开始

### 1. 克隆项目

```bash
git clone <repository-url>
cd ibreeze
```

### 2. 使用 Docker Compose (推荐)

```bash
# 启动 PostgreSQL
docker-compose up -d postgres

# 等待数据库就绪
sleep 5

# 安装后端依赖并运行迁移
cd apps/backend-api
pip install uv
uv sync
uv run alembic upgrade head

# 启动后端API
uv run uvicorn ibreeze_backend.main:app --reload --port 8000
```

### 3. 手动部署

#### 启动 PostgreSQL

```bash
# 创建数据库
psql -U postgres -c "CREATE DATABASE ibreeze;"
psql -U postgres -c "CREATE USER ibreeze WITH PASSWORD 'ibreeze';"
psql -U postgres -c "GRANT ALL PRIVILEGES ON DATABASE ibreeze TO ibreeze;"
```

#### 安装后端依赖

```bash
cd apps/backend-api
pip install uv
uv sync
```

#### 运行数据库迁移

```bash
uv run alembic upgrade head
```

#### 启动后端 API

```bash
uv run uvicorn ibreeze_backend.main:app --reload --port 8000
```

#### 启动管理后台

```bash
cd apps/admin-web
npm install
npm run dev
```

#### 启动桌面应用 (开发模式)

```bash
cd apps/desktop
npm install
npm run dev
```

#### 启动 Sidecar

```bash
cd sidecar
pip install uv
uv sync
uv run python -m ibreeze.main
```

## 配置

### 环境变量

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `IBREEZE_DATABASE_URL` | `postgresql+asyncpg://ibreeze:ibreeze@localhost:5432/ibreeze` | 数据库连接URL |
| `IBREEZE_API_KEY` | `dev-api-key-change-me` | API密钥 |
| `IBREEZE_TOKEN_SECRET` | `dev-token-secret-change-me` | JWT签名密钥 |

### Sidecar 端口

- 默认端口: `9527`
- 可通过 `--port` 参数修改

## 验证部署

### 检查后端 API

```bash
curl http://localhost:8000/health
# 期望输出: {"status":"ok"}

curl http://localhost:8000/health/ready
# 期望输出: {"status":"ready","database":"connected"}
```

### 检查管理后台

浏览器访问: http://localhost:5173

### 检查桌面应用

```bash
cd apps/desktop
npm run dev
```

## 常见问题

### 数据库连接失败

1. 确认 PostgreSQL 正在运行
2. 检查 `IBREEZE_DATABASE_URL` 配置
3. 确认用户和数据库已创建

### 端口冲突

- 后端 API: 8000
- 管理后台: 5173
- 桌面应用: 1420
- Sidecar: 9527

### 依赖安装失败

```bash
# 清除缓存重新安装
cd apps/backend-api
rm -rf .venv uv.lock
uv sync

cd apps/desktop
rm -rf node_modules package-lock.json
npm install
```
