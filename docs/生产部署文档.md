# 生产部署文档

## 概述

本文档描述 iBreeze AI 公司桌面应用的生产环境部署指南。

## 架构

```
                    ┌─────────────────┐
                    │   Load Balancer │
                    └────────┬────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
       ┌──────▼──────┐ ┌────▼─────┐ ┌──────▼──────┐
       │  Backend API │ │ Backend  │ │  Backend API │
       │   (Node 1)   │ │ (Node 2) │ │   (Node 3)   │
       └──────┬──────┘ └────┬─────┘ └──────┬──────┘
              │              │              │
              └──────────────┼──────────────┘
                             │
                    ┌────────▼────────┐
                    │   PostgreSQL    │
                    │   (Primary +    │
                    │    Replica)     │
                    └─────────────────┘
```

## 环境要求

### 服务器

| 组件 | 最低配置 | 推荐配置 |
|------|----------|----------|
| Backend API | 2 CPU, 4GB RAM | 4 CPU, 8GB RAM |
| PostgreSQL | 2 CPU, 4GB RAM | 4 CPU, 16GB RAM |
| Sidecar (per用户) | 1 CPU, 2GB RAM | 2 CPU, 4GB RAM |

### 软件版本

- **OS**: Ubuntu 22.04 LTS / CentOS 8+
- **Node.js**: 20.x LTS
- **Python**: 3.12
- **PostgreSQL**: 16
- **Redis**: 7.x (可选，用于缓存)

## Docker 部署

### 1. 构建镜像

```bash
# 构建后端 API
docker build -t ibreeze-backend-api:latest -f apps/backend-api/Dockerfile .

# 构建管理后台
docker build -t ibreeze-admin-web:latest -f apps/admin-web/Dockerfile .
```

### 2. Docker Compose (生产配置)

```yaml
version: '3.8'

services:
  postgres:
    image: postgres:16-alpine
    environment:
      POSTGRES_DB: ibreeze
      POSTGRES_USER: ibreeze
      POSTGRES_PASSWORD: ${DB_PASSWORD}
    volumes:
      - postgres_data:/var/lib/postgresql/data
    deploy:
      resources:
        limits:
          memory: 8G

  backend-api:
    image: ibreeze-backend-api:latest
    environment:
      IBREEZE_DATABASE_URL: postgresql+asyncpg://ibreeze:${DB_PASSWORD}@postgres:5432/ibreeze
      IBREEZE_API_KEY: ${API_KEY}
      IBREEZE_TOKEN_SECRET: ${TOKEN_SECRET}
    deploy:
      replicas: 3
      resources:
        limits:
          memory: 2G

  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx.conf:/etc/nginx/nginx.conf
      - ./certs:/etc/nginx/certs

volumes:
  postgres_data:
```

### 3. 启动服务

```bash
# 设置环境变量
export DB_PASSWORD=your-secure-password
export API_KEY=your-api-key
export TOKEN_SECRET=your-token-secret

# 启动所有服务
docker-compose -f docker-compose.prod.yml up -d
```

## 数据库配置

### 连接池配置

```ini
# PostgreSQL 配置
max_connections = 200
shared_buffers = 4GB
effective_cache_size = 12GB
work_mem = 16MB
```

### 备份策略

```bash
# 每日全量备份
pg_dump -U ibreeze ibreeze | gzip > backup_$(date +%Y%m%d).sql.gz

# 保留最近30天
find /backups -name "*.sql.gz" -mtime +30 -delete
```

## 安全配置

### SSL/TLS

```nginx
server {
    listen 443 ssl http2;
    ssl_certificate /etc/nginx/certs/fullchain.pem;
    ssl_certificate_key /etc/nginx/certs/privkey.pem;
    ssl_protocols TLSv1.2 TLSv1.3;
}
```

### 防火墙

```bash
# 只允许必要端口
ufw allow 80/tcp
ufw allow 443/tcp
ufw allow 22/tcp
ufw enable
```

### 密钥管理

- 使用环境变量或密钥管理服务
- 定期轮换 API 密钥和 JWT 密钥
- 禁用默认密码

## 监控

### 健康检查

```bash
# 后端 API
curl -f http://localhost:8000/health || exit 1

# 数据库
psql -U ibreeze -c "SELECT 1" || exit 1
```

### 日志收集

```bash
# 配置日志轮转
cat > /etc/logrotate.d/ibreeze << EOF
/var/log/ibreeze/*.log {
    daily
    rotate 30
    compress
    delaycompress
    missingok
    notifempty
}
EOF
```

### 指标监控

- **Prometheus**: 收集指标
- **Grafana**: 可视化监控
- **AlertManager**: 告警通知

## 扩展

### 水平扩展

```bash
# 增加后端 API 实例
docker-compose -f docker-compose.prod.yml up -d --scale backend-api=5
```

### 数据库读写分离

```ini
# 配置只读副本
DATABASE_READ_URL=postgresql+asyncpg://ibreeze:password@replica:5432/ibreeze
DATABASE_WRITE_URL=postgresql+asyncpg://ibreeze:password@primary:5432/ibreeze
```

## 回滚

```bash
# 回滚到上一版本
docker-compose -f docker-compose.prod.yml down
docker-compose -f docker-compose.prod.yml up -d --no-deps backend-api
```
